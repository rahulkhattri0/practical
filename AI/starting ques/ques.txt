Q7. Write a Python program to count the number 4 in a given list.
Sol.
def count_4(nums):
 c = 0
 for num in nums:
 if num == 4:
 c = c + 1
 return c
print(count_4([1, 2, 4, 5, 4]))
Output:
 
Q8. Write a Python program to compute the distance between the points (x1,

Sol.

x2 = int(input("Enter the value of x2 : "))

y2 = int(input("Enter the value of y2 : "))
